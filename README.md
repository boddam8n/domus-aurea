# Domus Aurea Luxury Wedding Platform

Next.js + TypeScript + Tailwind CSS + Framer Motion luxury wedding invitation platform.

## Run locally

```bash
npm install
npm run dev
```

Then open:

```text
http://localhost:3000
```

## Pages

- `/` home and platform showcase
- `/invitation/layan-yassin` personalized invitation
- `/rsvp` RSVP flow
- `/gallery` interactive gallery
- `/story` wedding story timeline
- `/contact` studio contact page
- `/dashboard` customer invitations and analytics
- `/auth/sign-in` and `/auth/sign-up`

## Notes

Static legacy files are still present for reference, but the new platform lives in the Next.js `app/`, `components/`, `lib/`, and `public/assets/` folders.
